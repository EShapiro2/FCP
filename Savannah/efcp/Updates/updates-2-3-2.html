    This script may be used to update the Logix and bin file content, starting in Efcp-release.


    pushd Logix/Bin
    cvs update
    cd ../system/Bin
    cvs update
    cd ../library_server/Bin
    cvs update
    cd ../../transform/Bin
    cvs update
    cd ../biospi/Bin
    cvs update
    cd ../../../Logix_Users/SpiFcp_Users/Bin
    cvs update
    cd ../BioSpi_Users/Bin
    cvs update
    popd


    Suggested method:

    cd Efcp-release
    copy the script to updater
    source updater
    Possibly due to a recent crash related to nongnu.org, you may see an error diagnostic, including:
    CVS update: conflicts found in spi_record.bin
    To correct this problem:
    cd Logix/system/Logix_Users/SpiFcp_Users/Bin
    rm spi_record.bin
    cvs update
