    The following shell script may be used to update the Logix and bin file content, starting in Efcp-release.


    pushd Logix/EMULATOR
    cvs update
    cd ../system/transform/biospi/Bin
    cvs update
    cd ../../../Logix_Users/SpiFcp_Users/Bin
    cvs update
    cd ../BioSpi_Users/Bin
    cvs update
    popd


    Suggested method:
    cd Efcp-release
    copy the script to: updater
    source updater
