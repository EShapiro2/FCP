Changes from SpiFcp/attributes.txt .A required attribute for Ambient Stochastic Pi Calculus modules is:    -language(biospi).The transformation, biospi, transforms modules written in AmbientStochastic Pi Calculus into modules in the compound dialect of LOGIX.An optional attribute is:    -mode(<execution mode>).where <execution mode> is one of:    interpret    interruptThe default mode is interrupt.This document is licensed under Gnu General Public License - Version 3     http://www.nongnu.org/efcp/gnu-gpl3.html
