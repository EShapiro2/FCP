Welcome to the Extended Flat Concurrent Prolog Download Page

To download the most recent release of the system, click efcp-release.tar.gz, and select Save (or Save it to Disk on Netscape). Then browse to the directory to which it is to be copied, and select Save.

To expand the release, enter:

    % zcat efcp-release.tar.gz | tar -xf -
A directory named Efcp-release and a system of sub-directories will be expanded in the current directory.

Cygwin, Linux and Macintosh users may expand the release by:

    % tar -zxf efcp-release.tar.gz

A release may be installed on cygwin, linux, maci386, macosx, sgi and solaris platforms. To install it on one or more of those processors, see Efcp-release/Logix/README.

Documentation for Logix, SpiFcp and BioSpi is at EFCP.

See BioSPI for infomation about the BioSPI Project.
