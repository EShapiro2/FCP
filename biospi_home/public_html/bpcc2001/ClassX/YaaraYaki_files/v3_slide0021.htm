Extract(MNO) ::=
       { Country ß head(MNO) } |
                        { MNO ß tail(MNO) } | Branch | Iterate.
Extract process:
Separates the list to two.
Country holds the head of the list,
MNO becomes the tail of the list.
(There’s a way to use a string instead of list by FCP operations…)
