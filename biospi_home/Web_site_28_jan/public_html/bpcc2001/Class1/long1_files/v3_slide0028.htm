Lewis’ Octet Theory
• The first 2, 10, 18 (etc) electrons form a
kernel or closed shell
• Those electrons outside a kernel but not
yet sufficient in number to form a full
octet are the valence electrons, e.g.
• Cl:    17 electrons – 10 kernel electrons  =
   7 valence electrons
• Sharing takes place so that both atoms can
have filled octets
